# Getting Started - Setup of your workspace

In this exercise, Please follow the instructions below to log into your SAP Build Code workspace using the credentials provided.

## Log in to SAP Build Code Lobby 🔐

1. Go to <a href="https://sap-btp-experience-2026-shared-sb-x94grxn8.us10.build.cloud.sap/lobby" target="_blank">SAP Build Lobby</a>

- This link will then open a login page.

<br>![](/exercises/setup/images/1_signin.png)

2. Enter the credentials, Each participant has been assigned a unique user ID for this hands-on session.

```
Credentials will be available at the event.
```

>where __XX__ is the two-digit number assigned to your workstation

3. Once you have entered the credentials, click on continue.
<br>![](/exercises/setup/images/2_sign_in.png)

4. You should now access SAP Build Lobby - your central workspace for creating and managing applications

<br>![](/exercises/setup/images/3_build.png)
